A Flickr plugin for Wordpress.

== Description ==

A Flickr plugin for Wordpress, built with [phpFlickr](http://phpflickr.com/).

You will need to get a [Flickr API key](http://www.flickr.com/services/api/keys/) and make sure to read the [Flickr API Terms of Service](http://www.flickr.com/services/api/tos/).

== Installation ==

1. Upload the 'my-flickr-gallery' folder to the `/wp-content/plugins/` directory.
2. Ensure the `cache` folder is writable by the webserver.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Configure the plugin through the 'Settings' menu in WordPress.
4. Place the `[mfg]` shortcode into a page or post.
